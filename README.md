# C# Console Calculator 🖥️💡

A simple yet **powerful console-based calculator** built in C# by **Arthur Hortmann**.  
This project demonstrates clean coding practices, input validation, and user-friendly console output with colors.

## Features ✨
- Supports arithmetic operations: `+`, `-`, `*`, `x`, `/`, `^`  
- Input validation with meaningful error messages  
- Colored console output:
  - **Red** for errors  
  - **Green** for successful results  
- Graceful exit with the command `"exit"`  
- Cleanly formatted results (2 decimal precision)  
- Beginner-friendly, ideal for learning **C# basics**, **console I/O**, and **control flow**
